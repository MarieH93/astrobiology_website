The Ultimate HTML Cheatsheet (this is a h1 heading)

The title

The name of the webpage you're on will be shown on the tab in your browser. This is specified in HTML as the head portion of the script, tagged as title, and comes before the body.

The formatting of Headings and Paragraphs (this is an h2 heading)

For every element in the HTML body, you can decide if it will be a heading or a paragraph.

With headings, there are 6 levels, tag h1 being the highest (largest print).

Paragraphs are tagged with a p instead. Giving each paragraph an opening and closing tag automatically separates them with a gap in between. The ps aren't numbered.

For more structure in your HTML file, you can also manually instert blank lines by using the br tag. The br tag is a special type of tag (called a 'void tag'); you don't need a closing br tag, just un opening one.


Presenting text in italics on the website requires the tag and closing tag before and after the text your want to write in italics.

Comments

You can write comments in your html code, which won't be displayed in the browser.

Lists

Html has two basic types of list:

  1. Ordered lists (<ol>) - displays a numbered list
  2. Unordered lists (<ul>) - displays a list of bullet points

Both have the same basic structure.

  • An opening list tag
  • Many list items (<li>)
  • A closing list tags

You don't often see lists in their basic form, but lists are heavily used in websites; dropdown menus are often cleverly styled lists.

Links

Links look like this: link to facebook. You specify where you want the link to point by specifying the 'href' inside the <a> tag.

The href can also point to local files. For example, this is a link to this page you're reading at the moment. Notice that the protocol and host part of the URL has been replaced by a "."

Images

Images are a bit like links. You specify the location of the image with the 'src' property in the <img> tag. Note that img tags are like br tags: they don't come in pairs. The 'alt' property should be used to provide a description of the image in case it won't load or the user is partially sighted.
